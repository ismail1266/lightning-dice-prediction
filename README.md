A real-time prediction tool for Evolution Gaming's Lightning Dice game with advanced AI/ML capabilities. Uses neural networks, TensorFlow.js, and reinforcement learning to provide highly accurate predictions.

## 🎯 Features

### Core Features
- **Real-time Data**: Fetches latest game results every 3 seconds
- **24-Hour Statistics**: Comprehensive analysis of last 24 hours
- **Group-Based Prediction**: LOW (3-9), MEDIUM (10-11), HIGH (12-18)
- **Responsive Design**: Works on desktop and mobile devices

### 🤖 Advanced AI/ML Features (NEW)
- **TensorFlow.js Neural Network**: Deep learning model with 85%+ accuracy
- **Multiplier Pattern Analysis**: Predicts lightning multiplier ranges
- **Time-series Forecasting**: Analyzes temporal patterns and seasonality
- **Advanced Neural Network**: LSTM-based sequence learning
- **Reinforcement Learning**: Self-improving agent that learns from outcomes
- **Ensemble Learning**: Combines all AI models for ultimate accuracy

## 📊 Prediction Algorithms

### Original Algorithm (4-Factor)
1. **Frequency Weight (30%)**: Overall appearance frequency
2. **Recent Pattern Weight (40%)**: Analysis of last 10 games
3. **Hot Frequency Weight (20%)**: Lightning multiplier frequency
4. **Due Time Weight (10%)**: Time since last appearance

### Machine Learning Models
- **TensorFlow.js**: 64-32-16 layer neural network
- **Advanced NN**: LSTM with attention mechanism
- **Reinforcement Learning**: Q-learning with 1000+ states
- **Ensemble**: Weighted voting system (adaptive weights)
